Downloading & Building From Source

The project services and source code repository for RvSnoop is hosted by SourceForge, you can get the latest version of the code by checking it out from the project repository at https://svn.sourceforge.net/svnroot/rvsn00p. The repository follows the standard trunk/branches/tags layout scheme, so to get the current working code get https://svn.sourceforge.net/svnroot/rvsn00p/trunk/rvsn00p.
