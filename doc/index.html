RvSnoop @version@ Documentation

This is the local documentatioon set for RvSnoop, there are two main components to the documentation:

User’s Guide
Most people should read this first, it contains instructions on using the software and tips on how to get the most benefit from it.
Developer’s Guide
For people interested in understanding more anout the internals of RvSnoop, extending it, or contributing patches and enhancements. This guide also has detailed information about the file formats used by RvSnoop, for those interested in interacting with them (e.g. reading record bundles).

If you're still having problems after reading those then you may be able to find an answer in the FAQ, there is a local copy or a possibly more up to date version on the web site.

If you’ve used RvSnoop before you can find a summary of changes in the latest version in the change log.

Stay Informed

You can keep up-to-date with new releases of RvSnoop by subscribing to the SourceForge file monitor. Also of interest is my weblog which frequently has posts about RvSnoop development.
