Welcome to the RvSnoop Website!

RvSnoop is an application that enables graphical monitoring of TIBCO Rendezvous™ messages. RvSnoop is designed to be used by developers and operational support staff and is easy to use and lightweight.

RvSnoop is cross platform and will run anywhere there is a suitable virtual machine. RvSnoop requires Java 1.4.2 or later (this will already be present if you have any of the TIBCO Runtime Agent based products installed).
